## Course Description

Building upon the foundational Machine Learning course, this advanced course delves deeper into sophisticated machine learning models and algorithms. Students will explore advanced techniques, including natural language processing, reinforcement learning, and generative models, and gain hands-on experience with cutting-edge tools and real-world applications.

## Objectives

- Implement and analyze advanced machine learning algorithms.
- Develop and optimize machine-learning models.
- Apply reinforcement-learning techniques to complex problems.
- Utilize generative models for data generation and augmentation.
- Conduct and present original research in machine learning.

## Prerequisites

- Introduction to Machine Learning
- Pattern Recognition
- Proficiency in Python programming

## Course Evaluation

- **Internal Examination:** 50%
- **Final Examination:** 50%

## Contents

### 1. Probabilistic Models and Inference [8 hours]

1.1 Probability distributions: Gaussian, Bernoulli, Multinomial, Dirichlet  
1.2 Maximum Likelihood Estimation (MLE)  
1.3 Maximum A Posteriori (MAP) estimation  
1.4 Bayesian inference: priors, posteriors, predictive distributions  
1.5 Bayesian model comparison  
1.6 Bayesian Inference and Parameter Estimation  
1.7 Bayesian regression  
1.8 Hierarchical Bayesian models  
1.9 Gaussian Processes and applications

### 2. Time Series Analysis [10 hours]

2.1 Introduction to Time Series Data  
2.2 Moving Averages, Window Function, Co-integration  
2.3 Autocorrelation, Auto-regression  
2.4 ARMA, ARIMA  
2.5 Recurrent Neural Networks (RNN, LSTM, GRU)

### 3. Anomaly Detection [7 hours]

3.1 Anomaly Detection Methods  
3.2 Isolation Forest  
3.3 Autoencoders for Anomaly Detection  
3.4 One-class Classification

### 4. Natural Language Processing and Neural Attention [10 hours]

4.1 Natural Language Processing  
4.2 Text Tokenization and Normalization  
4.3 Lemmatization and Stemming  
4.4 Text Embedding  
4.5 Applications of NLP  
4.6 Attention Mechanisms  
4.7 Types of Attention  
4.8 Transformer

### 5. Reinforcement Learning [10 hours]

5.1 Fundamentals of Reinforcement Learning  
5.2 Markov Model and Decision Processes  
5.3 Dynamic Programming (Value Iteration, Policy Iteration, Bellman Equation)  
5.4 Model Free Algorithms (Q-Learning, SARSA)  
5.5 Policy Gradient Methods (REINFORCE)  
5.6 Deep Reinforcement Learning (Introduction, Deep Q-Network)  
5.7 Practical Applications and Case Studies
